Responsive Tables

Yo can make your tables responsive. Firts add the class responsive and in all <td> add the propertie data-title with data title. Example:

<table class="responsive">
  <thead>
    <tr>
      <th>Singer</th>
      <th>Group</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td data-title="Singer">Freddie Mercury</td>
      <td data-title="Group">Queen</td>
    </tr>
    <tr>
     <td data-title="Singer">Kurk Cobain</td>
     <td data-title="Group">Nirvana</td>
    </tr>
    <tr>
      <td data-title="Singer">John Lennon</td>
      <td data-title="Group">Beatles</td>
      </tr>
    <tr>
      <td data-title="Singer">Jon Bon Jovi</td>
      <td data-title="Group">Bon Jovi</td>
    </tr>
  </tbody>
</table>

Utilities

MicroCSS include some functional classes:

  • text-left: Aling text to left
  • text-center: Centered text
  • text-left: Aling text to right
  • float-left: Add float-left to element
  • float-right: Add float-right to element
